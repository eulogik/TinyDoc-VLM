# Discord Community Setup Guide

## TinyDoc-VLM Discord Server

### Step 1: Create Server
1. Go to https://discord.com/create
2. Name: **TinyDoc-VLM**
3. Template: "Developers" or "Study Group"

### Step 2: Channels

```
📋
  #announcements     — Release notes, major updates
  #introductions     — New members introduce themselves
  #general           — General discussion
  #showcase          — Share what you're building

🤖
  #help              — Questions about using TinyDoc-VLM
  #model-discussion  — Architecture, training, benchmarks
  #deployment        — Edge deployment, ONNX, hardware
  #feature-requests  — What should we build next?

💻
  #development       — SDK, API, integrations
  #fine-tuning       — Domain-specific fine-tuning
  #data              — Training data, synthetic documents

🔧
  #bugs              — Bug reports
  #contributing      — PRs, issues, guidelines
```

### Step 3: Roles

| Role | Color | Description |
|------|-------|-------------|
| 🔮 Core Team | Gold | eulogik team members |
| 🧠 Researcher | Blue | ML researchers |
| 🛠️ Contributor | Green | Active contributors |
| 📦 Integrator | Purple | Building with TinyDoc-VLM |
| 💡 Early Adopter | Orange | Beta testers |

### Step 4: Welcome Message

```
Welcome to the TinyDoc-VLM community! 🎉

TinyDoc-VLM is the world's smallest document-specialist VLM (256M params).
Runs on Raspberry Pi, extracts structured data from documents, 100% open-source.

📖 Docs: https://github.com/eulogik/TinyDoc-VLM
🤗 Model: https://huggingface.co/eulogik/TinyDoc-VLM-256M
🐍 SDK: https://pypi.org/project/tinydoc/
🚀 Demo: https://huggingface.co/spaces/eulogik/TinyDoc-VLM

Pick a role in #roles, introduce yourself in #introductions, and dive in!
```

### Step 5: Bot Setup (Optional)

Recommended bots:
- **GitHub Bot** — Auto-post commits, PRs, issues
- **Welcome Bot** — DM new members with getting-started guide
- **Benchmark Bot** — Post benchmark results on merge to main

### Step 6: Invite Link

After setup, create invite: `https://discord.gg/TINYDISCORD`
Share in:

- HF model card
